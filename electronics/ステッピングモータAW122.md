# ステッピングモータ AW122

2026/03/20

Shigeichiro Yamasaki

## AW122 の特徴

* コンパクト設計: ボディの厚みが23mmと薄型（パンケーキ型）であるため、重量を抑えたい押出機部分への設置に最適です。
* 用途: 主に3DプリンターのTitan押出機、小型CNC機械、ロボット、自動化機器などに利用されます

## AW122 の仕様

* 製品名:	Iverntech 42 Nema 17 Stepper Motor 23mm Body
* ステップ角:	1.8°
* 相数：	2相 (4リード)
* 定格電圧：	4.1V
* 定格電流	1A/ 相
* 抵抗値： 4.1±10% Ω/相
* インダクタンス：	4.1±20% mH/相
* 保持トルク：	13N.cm 以上
* デデンとトルク：1.2N.cm
* 絶縁クラス	Bクラス
* サイズ: 42 × 42 × 23 mm（Nema 17規格）
* メーカー: Iverntech

## A4988 と Raspberry pi の接続


|A4988 ピン名|	Raspberry Pi / 電源|	補足説明|
|:--|:--|:--|
|VMOT	|外部電源 (+)	|12V〜24V（モーター駆動用）|
|GND (VMOT横) |	外部電源 (-)	|モーター電源のマイナス|
|VCC	|Raspberry Pi 3.3V|	ロジック回路用電源|
|GND (VCC横)	|Raspberry Pi GND|	ロジック用共通グランド|
|STEP	|GPIO 18 (Pin 12)	|パルス信号（1パルスで1ステップ）|
|DIR|	GPIO 23 (Pin 16)|	回転方向指定（High/Low）|
|ENABLE|	未接続 (またはGND)|	Lowで動作有効。通常は未接続でOK|
|MS1, MS2, MS3|	未接続 (または3.3V)|	マイクロステップ設定（後述）|
|RESET / SLEEP|	互いに接続|	この2つをジャンパ線で繋いでください|
|2B, 2A, 1A, 1B	|AW122 モーター|	モーターの4本線を接続（後述）|

### AW122 モーターの配線（4ピン）

AW122のコネクタ線は通常以下のペアになっています。A4988の端子に繋いでください。

    * A相ペア: 1A と 1B
    * B相ペア: 2A と 2B
  
    ※色がメーカーにより異なる場合がありますが、隣り合うピン同士がペアであることが多いです。

### 重要なポイント

* 電解コンデンサの挿入: VMOTとGNDの間に 100μF 以上の電解コンデンサ を必ず入れてください。これがないと電源投入時のスパイク電圧でA4988が即死します。
* 電流制限（Vref）の調整: 通電前にA4988の小さなネジ（ボリューム）を回し、テスターで Vrefを 0.4V〜0.6V 程度 に合わせてください（1.0Aを超えないようにするため）。
* マイクロステップの設定: MS1〜MS3をすべて未接続（Low）にすると「フルステップ（1回転200分割）」になります。すべて3.3Vに繋ぐと「1/16ステップ」になり、非常に静かになります。
