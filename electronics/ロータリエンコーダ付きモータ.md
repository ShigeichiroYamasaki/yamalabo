## モータドライバ L298 の場合

* Raspberry pi Zero と接続

### JGA23-371 の配線

*	•	赤: モータ +
*	•	白: モータ -
*	•	黄: Encoder A
*	•	緑: Encoder B
*	•	青: Encoder VCC
*	•	黒: Encoder GND

### JGA25-371 と L298/Raspberry pi との接続

注意：Encoder VCC は 3.3V に接続する

*	•	赤: モータ + -->  L298N のモータ出力
*	•	白: モータ - -->  L298N のモータ出力
*	•	黄: Encoder A --> GPIO17
*	•	緑: Encoder B --> GPIO18
*	•	青: Encoder VCC --> 3.3V
*	•	黒: Encoder GND --> GND

#### L298 と Raspberry pi との接続

* •	ENA → Pi --> GPIO12(PWM)
*	•	IN1 → Pi --> GPIO20
*	•	IN2 → Pi --> GPIO21


## プログラム

```python
import RPi.GPIO as GPIO
import time

# ----- Motor pins -----
ENA = 12
IN1 = 20
IN2 = 21

# ----- Encoder pins -----
ENC_A = 17   # yellow
ENC_B = 18   # green

count = 0

GPIO.setmode(GPIO.BCM)
GPIO.setwarnings(False)

GPIO.setup(ENA, GPIO.OUT)
GPIO.setup(IN1, GPIO.OUT)
GPIO.setup(IN2, GPIO.OUT)

# エンコーダ入力
# Hall出力がオープンコレクタ気味の個体対策として pull_up を入れる
GPIO.setup(ENC_A, GPIO.IN, pull_up_down=GPIO.PUD_UP)
GPIO.setup(ENC_B, GPIO.IN, pull_up_down=GPIO.PUD_UP)

pwm = GPIO.PWM(ENA, 1000)
pwm.start(0)

def motor_forward(speed=60):
    GPIO.output(IN1, GPIO.HIGH)
    GPIO.output(IN2, GPIO.LOW)
    pwm.ChangeDutyCycle(speed)

def motor_backward(speed=60):
    GPIO.output(IN1, GPIO.LOW)
    GPIO.output(IN2, GPIO.HIGH)
    pwm.ChangeDutyCycle(speed)

def motor_stop():
    pwm.ChangeDutyCycle(0)
    GPIO.output(IN1, GPIO.LOW)
    GPIO.output(IN2, GPIO.LOW)

def encoder_callback(channel):
    global count
    a = GPIO.input(ENC_A)
    b = GPIO.input(ENC_B)

    # A相の立上り時にB相を見て方向判定
    if a == 1:
        if b == 0:
            count += 1
        else:
            count -= 1

GPIO.add_event_detect(ENC_A, GPIO.RISING, callback=encoder_callback, bouncetime=1)

try:
    count = 0
    motor_forward(60)

    start = time.time()
    while time.time() - start < 5:
        print(f"count = {count}")
        time.sleep(0.2)

    motor_stop()
    print("final count =", count)

finally:
    pwm.stop()
    GPIO.cleanup()
```
