# バイポーラ・ステッピングモータドライバ A4988

2026/03/18

Shigeichiro Yamasaki

2相バイポーラステッピングモータ用ドライバ

## 利点

* 定電流制御（チョッパ制御）
A4988は、モーターに流れる電流を常に監視し、設定した電流（AW122なら1.0A）を超えないように自動調整します。これにより、12Vや24Vといった高電圧をかけてもモーターが焼き切れる心配がなく、高速・高トルクで回せます。
* マイクロステップ駆動
1回転を200分割（1.8度ごと）するだけでなく、最大で1/16（3200分割）まで細かく制御できます。これにより、動作が非常に滑らかになり、騒音も抑えられます。
制御が非常にシンプル
* マイコン（Arduino等）からは「STEP（一歩進む）」と「DIR（回転方向）」の2本の信号線だけで操作可能です。

## 仕様

* Vcc: 2~7V
* VM モータ電圧： 0~11V
* H ブリッジ電流： 0 ~ 1.5A
* PWM 周波数： 0~250kHz
* VIN ロジック入力電圧： 0 ~ Vcc
