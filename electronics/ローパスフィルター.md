## 移動平均法 (Moving Average Method)

$$ y(t) = \frac{1}{k}\sum_{i=0}^{k-1}x(t-i) $$

```python
def LPF_MAM(x,times,tau = 0.01):
    k = np.round(tau /(times[1] - times[0])).astype(int)
    x_mean =  np.zeros(x.shape)
    N = x.shape[0]
    for i in range(N):
        if  i-k//2 <0 :
            x_mean[i]  = x[: i - k//2 +k].mean()
        elif i - k//2 +k>=N:
            x_mean[i]  = x[i - k//2 :].mean()
        else :
            x_mean[i]  = x[i - k//2 : i - k//2 +k].mean()
    return x_mean


# 正弦波のプロット
x_MAM = LPF_MAM(x,times,0.035)
plt.plot(times, x, label="sin")
plt.plot(times, x_MAM, label="sin MEM")
plt.legend()
plt.show()

# 矩形波のプロット
y_MAM = LPF_MAM(y,times,0.03)
plt.plot(times, y, label="step")
plt.plot(times, y_MAM, label="step MEM")
plt.legend()
plt.show()
```

## ガウス畳み込み

* 平均 0, 分散 $\sigma^2$ ガウス関数
$$ g_\sigma(t) = \frac{1}{\sqrt{2\pi \sigma^2}}\exp\Big(\frac{t^2}{2\sigma^2}\Big) $$

* ローパスフィルター

$$y(t) =  (g_\sigma*x)(t) = \sum_{i=-n}^n g_\sigma(i)x(t+i)$$

*  $n=3\sigma$ とする

```python
def  LPF_GC(x,times,sigma):
    sigma_k = sigma/(times[1]-times[0]) 
    kernel = np.zeros(int(round(3*sigma_k))*2+1)
    for i in range(kernel.shape[0]):
        kernel[i] =  1.0/np.sqrt(2*np.pi)/sigma_k * np.exp((i - round(3*sigma_k))**2/(- 2*sigma_k**2))
        
    kernel = kernel / kernel.sum()
    x_long = np.zeros(x.shape[0] + kernel.shape[0])
    x_long[kernel.shape[0]//2 :-kernel.shape[0]//2] = x
    x_long[:kernel.shape[0]//2 ] = x[0]
    x_long[-kernel.shape[0]//2 :] = x[-1]
        
    x_GC = np.convolve(x_long,kernel,'same')
    
    return x_GC[kernel.shape[0]//2 :-kernel.shape[0]//2]

sigma = 0.011


# 正弦波のプロット
x_GC = LPF_GC(x,times,sigma)
plt.plot(times, x, label="sin")
plt.plot(times, x_GC, label="sin GC")
plt.legend()
plt.show()

# 矩形波のプロット
y_GC = LPF_GC(y,times,sigma)
plt.plot(times, y, label="step")
plt.plot(times, y_GC, label="step MEM")
plt.legend()
plt.show()
```

## 一次遅れ系

* カットオフする周波数基準: $f_0$

* 離散方程式
$$ y(t+1) = \Big(1 - \frac{\Delta t}{f_0}\Big)y(t) + \frac{\Delta t}{f_0}x(t)$$

```python
def LPF_F0(x,times,f_F0=10):
    x_F0 = np.zeros(x.shape[0])
    x_F0[0] = x[0]
    dt = times[1] -times[0]
    for i in range(times.shape[0]-1):
        x_F0[i+1] =  (1-  dt*f_F0) *x_F0[i]  + dt*f_F0* x[i]
    return x_F0

f0 = 100
# 正弦波のプロット
x_F0 = LPF_F0(x,times,100)
plt.plot(times, x, label="sin")
plt.plot(times, x_F0, label="sin F0")
plt.legend()
plt.show()

# 矩形波のプロット
y_F0 = LPF_F0(y,times,100)
plt.plot(times, y, label="step")
plt.plot(times, y_F0, label="step F0")
plt.legend()

```

### Pythonによるローパスフィルターの実装 (Butterworthフィルタ)

* scipy.signal.butter: フィルターの係数（
）を計算します。btype='low'でローパスを指定します。
* fc / (fs / 2): カットオフ周波数はナイキスト周波数（サンプリング周波数の半分）で正規化する必要があります。
* scipy.signal.filtfilt: 信号に対してフィルターを適用します。lfilterよりも位相遅れが発生しないため、データ分析ではfiltfiltが好まれます。


```python
import numpy as np
import matplotlib.pyplot as plt
from scipy import signal

# --- 1. サンプルデータの作成 (ノイズを含む信号) ---
fs = 1000  # サンプリング周波数 (Hz)
t = np.linspace(0, 1, fs)  # 1秒間の時間軸
# 5Hzの信号 + 高周波ノイズ
sig = np.sin(2 * np.pi * 5 * t) + 0.5 * np.random.normal(size=len(t))

# --- 2. ローパスフィルターの設計 ---
fc = 10  # カットオフ周波数 (Hz) - これより高い周波数を減衰させる
order = 5  # フィルタの次数

# バターワースフィルタの係数を計算（ローパス）
b, a = signal.butter(order, fc / (fs / 2), btype='low')

# --- 3. フィルターの適用 ---
filtered_sig = signal.filtfilt(b, a, sig)  # ゼロ位相フィルタリング

# --- 4. 結果の描画 ---
plt.figure(figsize=(10, 6))
plt.plot(t, sig, label='Original with Noise', alpha=0.5)
plt.plot(t, filtered_sig, label='Lowpass Filtered (10Hz)', linewidth=2)
plt.legend()
plt.grid()
plt.show()
```
