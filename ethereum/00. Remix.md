## Remix IDE

* [Remix IDE サイト](https://remix.ethereum.org)  https://remix.ethereum.org

* デスクトップ版もあります．
* Remix IDE には，Remix VM という内蔵 EVM を備えているので，コントラクトをそこにデプロイしてテストなどを行うことができます．

### Remix IDE の左サイドバー

* AI
* File Explorer（ファイル管理）
* Solidity コンパイラ
* デプロイと実行
* デバッガー
* Slidity 単体テスト
* プラグイン管理

### 中央

* 中央：コードエディター
* 下：実行ログ／エラー／デバッグ情報

## Remix ワークスペースとプロジェクト

Remix IDE におけるプロジェクト単位の仮想ファイル・システムを「ワークスペース」と呼びます．

Remix IDE では，プロジェクトごとに，ワークスペースを単位として，コントラクトのコーディング，テスト，デプロイ，関連ライブラリなどをまとまりとして管理します．

### ワークスペースの新規作成

![](https://remix-ide.readthedocs.io/en/latest/_images/a-fe-workspaces-new.png)

テンプレートの一覧から選択する


* Remix の一般的テンプレート
* OpenZeppelinテンプレート(ERC20、ERC721、ERC1155)
* Cookbook.devテンプレート
* Gnosis Safe MultiSig テンプレート
* Zero-Knowledge Proof (ZKP) テンプレート(Circom、Noir、Generic)
* Uniswap v4テンプレート
* Solidity CREATE2 テンプレート
* コントラクト確認のための Etherscan スクリプト
* ユニットテストおよびセキュリティ分析のためのCIスクリプト
