# 2. Hardhat による Ethereum ブロックチェーンのコンソール操作

最終更新 2024/10/30 Shigeichiro Yamasaki

## 前提条件

* 自分の Hardhat プロジェクトのプロジェクトルートにいることを前提にします．
* localhost で Hardhat node が稼働している
* コントラクトが Hardhat node にデプロイ済


## hardhat console を起動して hardhat node に接続する

hardhat console コマンドで node のプロンプトになります

```bash
npx hardhat console --network localhost 

=>
Type ".help" for more information.
>
```

### config オブジェクトの確認

## コンソールからの操作

### コントラクトアドレス

デプロイしたときにログとして表示されたコントラクトアドレスを定数に代入しておく

```js
const contract_addr = "0x5fbdb2315678afecb367f032d93f642f64180aa3"

```

### デプロイされているコントラクトとの接続

hardhat network ノードのログをチェックし，トランザクションの発出などを見ておく

```js
// デプロイしたコントラクト(Token)のコントラクトアドレス
> const contract_addr = "0x5fbdb2315678afecb367f032d93f642f64180aa3"
// コントラクトの ABI, バイトコードなどを取得する
> const factory = await ethers.getContractFactory("Token")
// 当該コントラクトアドレスのコントラクトに接続する
> const contract = await factory.attach(contract_addr)
```

### トークン残高の確認

状態変数の参照

> await contract.balanceOf(owner.address);

### トークンの送金(addr1 → addr2)

```js
> await contract.connect(addr1).transfer(addr2.address, 50)

> await contract.balanceOf(addr1.address);

> await contract.balanceOf(addr2.address);
50n
```
