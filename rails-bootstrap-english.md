## How to use Rails bootstrap

### Layout modification

`` `bash
nano app / views / layouts / application.html.erb
`` ```


Add a div tag to the body tag part as shown below (enclose <% = yield%> in div)

`` `html
  ...

  <body>
    <div class = "container">
    <% = yield%>
    </ div>
  </ body>
  
...
`` ```

### Server restart


`` `bash
rails s -b 0.0.0.0
`` ```


### Reload the browser and check the layout change
